# Telegram Games Bot

Простой Telegram-бот с мини-играми, виртуальной валютой и ставками. Хранит баланс и историю партий в SQLite.

## Игры

Все игры на ставку.

| Игра | Описание |
|---|---|
| 🎯 Угадай число | Бот загадывает 1..100, лимит 10 попыток, выплата зависит от скорости |
| ✊ Камень-ножницы-бумага | Классическая КНБ против бота, победа ×2 |
| 🪙 Монетка | Орёл или Решка, выигрыш ×2 |
| 🎲 Кости | Больше/Меньше ×2 или точное число ×6 |

## Установка

```bash
pip install telebot
```

SQLite встроена в стандартную библиотеку Python — отдельно ставить не нужно.

## Настройка

1. Получи токен у [@BotFather](https://t.me/BotFather) (`/newbot`).
2. Замени значение `TOKEN` в [test.py](test.py).

> ⚠️ Текущий токен в файле — тестовый и публичный. Перед продакшеном отзови его в @BotFather и подставь свой.

## Запуск

```bash
python test.py
```

База данных `bot.db` создастся автоматически при первом запуске.

## Команды

| Команда | Действие |
|---|---|
| `/start` | Приветствие + главное меню |
| `/menu` · `/play` | Меню игр |
| `/balance` | Текущий баланс |
| `/stats` | Статистика: партии, W/L, винрейт, чистый итог |

## Экономика

- Стартовый баланс: **1000 💰**
- Ставки: `10`, `50`, `100`, `500` (недоступные скрываются)
- При балансе `0` появляется кнопка «💰 Пополнить +1000»

### Выплаты

| Ставка | Шанс | Коэффициент | EV |
|---|---|---|---|
| Монетка (орёл/решка) | 50% | ×2 | 0 |
| КНБ (без ничьей) | 50% | ×2 | 0 |
| Кости: больше/меньше | 50% | ×2 | 0 |
| Кости: точное число | ~16.7% | ×6 | 0 |
| Угадай число — 1 попытка | 1% | +10× | высокий |
| Угадай число — 2–3 попытки | ~4% | +3× | — |
| Угадай число — 4–6 попыток | выше | +1× | — |
| Угадай число — 7–10 | — | возврат | 0 |
| Угадай число — не угадал | — | −ставка | — |

## Структура БД

`users`
| поле | тип | описание |
|---|---|---|
| `user_id` | INTEGER PK | Telegram user id |
| `username` | TEXT | @username |
| `balance` | INTEGER | текущий баланс |
| `created_at` | TEXT | дата создания |

`game_history`
| поле | тип | описание |
|---|---|---|
| `id` | INTEGER PK | |
| `user_id` | INTEGER FK | → `users.user_id` |
| `game` | TEXT | `coin` / `dice` / `dice_exact` |
| `bet` | INTEGER | размер ставки |
| `result` | TEXT | `выбор/итог`, напр. `heads/tails` |
| `delta` | INTEGER | изменение баланса (+/−) |
| `played_at` | TEXT | timestamp |
